# Shopping List App

An Android application that allows users to create, manage, and organize shopping lists, similar in concept to Google Keep.

## Overview

This project is being developed as part of a work-study course in computer science. The goal of the project is to design and implement a functional Android application while applying core software development principles, including object-oriented programming, user interface design, and version control using GitHub.

The application focuses on simplicity and usability, allowing users to quickly add, view, and manage items in a shopping list.

## Features

- Add items to a shopping list
- Remove items from the list
- View all items in a simple, clean interface
- Designed for ease of use on Android devices

### Planned Features

- Edit existing items
- Multiple shopping lists
- Persistent data storage
- Improved UI and accessibility features

## Technologies Used

- Java
- Android Studio
- Android SDK
- Git & GitHub
